   • Confirm the security warning

STEP 2: Download APK
   • Get app-debug.apk from:
     ├─ app/build/outputs/apk/debug/ (if built locally)

     └─ Direct download link provided

STEP 3: Transfer to Device
   Option A: USB Cable
      • Connect device via USB
      • Copy app-debug.apk to Downloads folder
   
   Option B: Cloud Storage
      • Upload APK to Google Drive/Dropbox
      • Download on device
   
   Option C: Direct ADB
      $ adb install app/build/outputs/apk/debug/app-debug.apk

STEP 4: Install APK
   • Open File Manager
   • Navigate to Downloads folder
   • Tap app-debug.apk
   • Tap "Install"
   • Wait 30-60 seconds for installation
   • Tap "Done" when complete

STEP 5: Grant Permissions
   First launch will request permissions:
   ✓ Camera - For jewelry photo capture
   ✓ Storage - For file access and database
   ✓ Internet - For Gemini AI and Firebase
   
   Tap "Allow" for each permission

STEP 6: Launch App
   • Find "Swarnaly Shilpaloy" in app drawer
   • Tap to launch
   • App should open without errors
   • Complete initial setup

🐛 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

❌ Error: "Build fails with SDK error"
✅ Solution:
   • Open SDK Manager in Android Studio
   • Download Android SDK level 36
   • Ensure Build Tools 36.0.0 is installed
   • Retry build

❌ Error: "GEMINI_API_KEY not found"
✅ Solution:
   • Create .env file in project root
   • Add: GEMINI_API_KEY=your_key_here
   • Save and rebuild

❌ Error: "APK won't install on device"
✅ Solution:
   • Check Android version is 7.0 or higher
   • Enable "Unknown Sources" in Settings
   • Free up storage space
   • Try uninstalling previous version first

❌ Error: "App crashes immediately after launch"
✅ Solution:
   • Ensure internet connection is active
   • Check if Gemini API key is valid
   • Clear app cache: Settings → Apps → Swarnaly → Storage → Clear Cache
   • Restart device
   • Reinstall APK

❌ Error: "Gradle build extremely slow"
✅ Solution:
   • Close other applications
   • Enable gradle caching
   • Use: ./gradlew --daemon
   • Increase Gradle heap: -Xmx4g in gradle.properties
