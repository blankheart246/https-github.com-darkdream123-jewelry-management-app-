🔧 TROUBLESHOOTING

Fix: Create .env file with valid API key

Issue: APK won't install
Fix: Check min SDK requirement (24+), clear cache

Issue: App crashes at startup
Fix: Check logcat, verify internet, check Firebase config

Issue: Gradle build slow
Fix: Enable caching, use gradle.properties parallel=true
